# Resumo do trabalho

Este trabalho investiga estratégias para prolongar o período de operação com redundância
funcional em redes IIoT (Industrial Internet of Things) destinadas ao monitoramento de
variáveis críticas em ambientes de difícil manutenção. A proposta combina ESP-NOW na
comunicação intra-cluster e Wi-Fi e MQTT (Message Queuing Telemetry Transport) no
uplink realizado pelo líder, cuja responsabilidade é alternada pelas políticas Round-Robin,
energia residual e energia residual com cooldown. A avaliação utiliza simulação em Python
e um protótipo utilizando a placa de desenvolvimento ESP32-C3 SuperMini. O estudo
organiza a base teórica e metodológica para comparar políticas de liderança, analisar falhas
de comunicação e verificar o consumo energético por papel operacional de cada nó na rede.
